# Browser Minimap

A Tampermonkey userscript that adds a VSCode-style minimap to any web page.

## Features

- Minimal, fast layout-only minimap in Shadow DOM
- Follows hovered scrollable areas
- Toggle with 'M' key
- Click/drag to scroll
- Number keys: 1 (top 0%), 0 (bottom 100%)
- Hydration-safe for React/Next.js sites
- Excludes common sites like YouTube, Twitter

Install via Tampermonkey from the main.js file.
