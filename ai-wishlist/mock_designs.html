Settings Interface & Extension Options

We need to add a dedicated Settings view for storing API keys, choosing models, themes, accent colors, and toggling background scraping. Play with the options below to pick the best flow.

1 Select Settings UI Layout
Option A: Slide-Over Drawer Sheet
Settings panel glides in smoothly from the right, overlaying the active wishlist. Header contains an elegant "Back" arrow to return. Excellent for focus and clean workspace.
Option B: 3D Flip Card Panel
The entire panel rotates 180 degrees using CSS 3D perspectives to show settings on the back-face. Playful, premium, and highly engaging micro-animation.
Option C: Inline Tab Switcher
Adds a standard top tab navigation bar (Items / Settings). Smoothly fades settings controls in place of the product list, retaining consistent header height.
Option D: HUD Glass Modal
Settings open inside a centered floating card overlay, blurring the product rows in the background. Great separation of concerns.
💡 Try it Out!

Use the preview panel on the right. Click the gear icon (⚙) in the wishlist header to trigger settings. Choose accent dots or theme dropdowns to see styling variables apply dynamically!
